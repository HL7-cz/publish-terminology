Allergy Intolerance Category (CZ Code system supplement)

This Code system is referenced in the content logical definition of the following value sets:

  • AllergyIntoleranceCategory

This code system supplement https://hl7.cz/terminology/CodeSystem/allergy-intolerance-category-cz defines displays on the following codes:

CodeDisplayDefinitionEnglish (English, en)
biologic biologický Přípravek syntetizovaný z živých organismů nebo jejich produktů, zejména lidských nebo živočišných bílkovin, jako jsou hormony nebo antitoxiny, který se používá jako diagnostické, preventivní nebo terapeutické činidlo. Mezi biologické léky patří například: vakcíny; alergenní extrakty, které se používají jak k diagnostice, tak k léčbě (například alergenové injekce); genové terapie; buněčné terapie. Existují i jiné biologické produkty, jako jsou tkáně, které obvykle nejsou spojovány s alergiemi. A preparation that is synthesized from living organisms or their products, especially a human or animal protein, such as a hormone or antitoxin, that is used as a diagnostic, preventive, or therapeutic agent. Examples of biologic medications include: vaccines; allergenic extracts, which are used for both diagnosis and treatment (for example, allergy shots); gene therapies; cellular therapies. There are other biologic products, such as tissues, which are not typically associated with allergies.
environment prostředí Jakékoli látky, které se vyskytují v životním prostředí, včetně látek, které dosud nebyly klasifikovány jako potraviny, léky nebo biologické látky. Any substances that are encountered in the environment, including any substance not already classified as food, medication, or biologic.
food potravina Jakákoli látka konzumovaná za účelem výživové podpory organismu. Any substance consumed to provide nutritional support for the body.
medication léčivo Látky podávané za účelem dosažení fyziologického účinku. Substances administered to achieve a physiological effect.
