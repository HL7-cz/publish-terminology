Language: cs

This value set includes codes based on the following rules:

  • Import all the codes that are contained in Typy dokumentů obrazového komplementu
  • Import all the codes that are contained in Typ ošetřovatelské dokumentace
  • Import all the codes that are contained in Typy žádanek
  • Import all the codes that are contained in Typy laboratorních dokumentů
  • Import all the codes that are contained in Typy lékařské dokumentace
